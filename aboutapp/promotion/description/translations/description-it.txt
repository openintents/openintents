# Application:
OI Info

# Promo text:
Semplice finestra di informazioni condivisa tra le applicazioni.

# Description:
OI About è un dialogo di base il quale è condiviso fra le applicazioni. Questa applicazione è invocata attraverso il menu di altre applicazioni OI. Vi permette di visualizzare l'informazione della versione, i link, i crediti, e la licenza dell'applicazione. I crediti includono gli autori, le documentazioni, i traduttori, e gli artisti. Tutte queste funzioni sono portate assieme in una struttura semplice e efficiente. 

Questa applicazione non contiene la pubblicità e non richiede la permissione di internet. 

Il codice sorgente dell' applicazione gratuita e con la sorgente aperta è disponibile a: 
http://code.google.com/p/openintents

Per la lista completa con le modifiche e una lista con le domande frequenti, prego visitare: 
http://www.openintents.org

Potete aiutare a migliorare la traduzione nella vostra lingua a Launchpad: 
https://translations.launchpad.net/openintents/trunk

Caratteristiche: 
* Mostra le informazioni generali, i crediti, e i file con licenza.
* Mostra le traduzioni per tutte le versioni della lingua.
* Recupera l'informazione del manifesto dell'applicazione attraverso metadata e marche implicite.
* Opzionale, tutti i campi possono essere modificati con l'aiuto di extra-intenzione.

Le applicazioni che funzionano con "OI About": 
* OI Annotazioni
* OI Documenti manager
* Lista di acquisti OI
* OI converte CVS
* OI Sicurezza
* OI Conto alla rovescia
