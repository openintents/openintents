# Application:
OI Σχετικά

# Promo text:
Ένας αυτόνομος σχετικός διάλογος που μοιράζεται μεταξύ των εφαρμογών.

# Description:
Το OI Σχετικά είναι ένας αυτόνομος σχετικά διάλογος που διαμοιράζεται μεταξύ των εφαρμογών. Αυτή η εφαρμογή εφαρμόζεται μέσω των μενού άλλων OI εφαρμογών. Σας επιτρέπει να προβάλλετε πληροφορίες έκδοσης, συνδέσμους, συντελεστές και την άδεια μιας εφαρμογής. Οι συντελεστές περιλαμβάνουν συγγραφείς, άτομα στην τεκμηρίωση, μεταφραστές και καλλιτέχνες. Όλη αυτή η λειτουργικότητα βρίσκεται κοντά σε μια απλά αλλά επαρκή δομή. 

Αυτή η εφαρμογή δεν έχει διαφημήσεις και δεν απαιτεί άδεια στο διαδίκτυο. 

Ο πηγαίος κώδικας αυτού είναι δωρεάν και η εφαρμογή ανοικτού κώδικα είναι διαθέσιμη στο: 
http://code.google.com/p/openintents

Για την πλήρη λίστα των αλλαγών και μια λίστα συχνών ερωτημάτων, παρακαλώ επισκευτείτε: 
http://www.openintents.org

Μπορείτε να βοηθήσετε στη βελτίωση της μετάφρασης στη γλώσσα σας στο Launchpad: 
https://translations.launchpad.net/openintents/trunk

Χαρακτηριστικά: 
* Εμφανίζει γενικές πληροφορίες, συντελεστές και καρτέλες άδειας.
* Εμφανίζει τους μεταφραστές όλων των γλωσσών.
* Ανακτά πληροφορίες από το μανιφέστο της εφαρμογής μέσω των προεπιλεγμένων ετικετών και των μεταδεδομένων.
* Προαιρετικά, όλα τα πεδία μπορούν να τροποποιηθούν μέσω επιπρόσθετων προθέσεων.

Οι εφαρμογές που συνεργάζονται με το "OI Σχετικά": 
* OI Σημειωματάριο
* OI Διαχειριστής Αρχείων
* OI Λίστα Αγορών
* OI Μετατροπέας CSV
* OI Κιβώτιο
* OI Χρονομετρητής
* OI Φακός

Λέξεις κλειδιά:

* OI, OpenIntents, Open Intents.
