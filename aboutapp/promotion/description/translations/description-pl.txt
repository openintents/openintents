# Application:
OI About

# Promo text:
Dialog informacyjny dzielony miedzy aplikacjami

# Description:
"Ol About" to samodzielny dialog, który jest dzielony między aplikacjami. Ta aplikacja jest wywoływana za pomocą menu innych aplikacji Ol. Umożliwia wyświetlanie informacji o wersji, linków, punktów i licencji aplikacji. Punkty obejmują autorów dokumentacji, tłumaczy i artystów. Wszystkie te funkcje są połączone w uproszczony i efektywny układ. 

Ta aplikacja jest darmowa i nie wymaga dostępu do internetu. 

Kod źródłowy tej darmowej i open-source-owej aplikacji jest dostępny pod adresem: 
http://code.google.com/p/openintents

Aby zobaczyć pełną listę zmian oraz FAQ, odwiedź: 
http://www.openintents.org

Możesz poprawić tłumaczenie na Twój język: 
https://translations.launchpad.net/openintents/trunk

Cechy: 
* Pokazuje ogólne informacje,punkty i karty licencji.
* Pokazuje tłumaczy wszystkich wersji językowych.
* Pobiera informacje z manifestu aplikacji poprzez domyślne etykiety i metadane.
* Opcjonalnie wszystkie pola mogą być modyfikowane poprzez dodatkowe zamiary.

Aplikacja który pracują z "Ol About": 
* Notatnik Ol
* Menedżer plików Ol
* Lista zakupów Ol
* OI Konwertuj CSV
* Bezpieczne Ol
* OI Odliczanie
* OI Latarka
