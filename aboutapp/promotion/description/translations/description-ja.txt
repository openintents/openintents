# Application:
OI About

# Promo text:
アプリ間で共有されるダイアログについての単体アプリ

# Description:
OI 情報はアプリケーションの間で共有されるスタンドアローン情報ダイアログです。 このアプリケーションは他の OI アプリケーションのメニューから呼び出されます。 アプリケーションのバージョン情報、リンク、クレジットやライセンスを見ることができます。 クレジットは著者、ドキュメンテーション、翻訳者やアーティストを含みます。 すべての機能はシンプルで効率的なレイアウトで集結されています。 

このアプリケーションは広告を含まず、インターネット許可を必要としません。 

この無料でオープンソースのアプリケーションのソースコードは以下の場所から取得できます： 
http://code.google.com/p/openintents

すべての変更リストや質問リストを見るには： 
http://www.openintents.org

あなたもこの言語の翻訳を Launchpad で改善することができます： 
https://translations.launchpad.net/openintents/trunk

機能： 
* 一般情報、クレジットとライセンスタブを表示します。
* すべての言語の翻訳者を表示します。
* アプリケーションマニフェストから、デフォルトタグやメタデータを通して情報を入手します。
* すべてのフィールドはインテント エクストラ から任意で変更することができます。

"OI 情報" と動作するアプリケーション： 
* OI メモ帳
* OI ファイルマネージャー
* OI ショッピングリスト
* OI CSV 変換
* OI 金庫
* OI カウントダウン
* OI フラッシュライト

キーワード: 
* 情報とクレジット
* OI, OpenIntents, Open Intents.
