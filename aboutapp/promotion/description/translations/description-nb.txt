# Application:
OI Info

# Promo text:
En enkeltstående informasjonsdialog som er delt mellom applikasjoner.

# Description:
OI Info er en frittstående informasjonsdialog som er delt mellom apper. Denne applikasjonen blir påkalt gjennom menyen til andre OI applikasjoner. Den lar deg se på versjonsinformasjon, linker, krediteringer og lisensen til en applikasjon. Krediteringer inkluderer forfattere, dokumentasjon, oversettere og kunstnere. All denne funksjonaliteten er ført sammen i en enkel og effektiv utforming. 

Denne applikasjonen er reklamefri og trenger ikke inter 

Kildekoden til denne gratis applikasjonen med åpen kildekode er tilgjengelig hos: 
http://code.google.com/p/openintents

For den komplette listen med endringer og en liste over vanlige spørsmål, vennligst besøk: 
http://www.openintents.org

Du kan hjelpe til med å forbedre oversettelsen 
https://translations.launchpad.net/openintents/trunk

Funksjoner: 
* Viser generell informasjon, krediteringer og lisensfaner.
* Viser oversettere av alle språkversjoner.
* Henter informasjon fra applikasjonsmanifest gjennom standardmerker og metadata.
* Alternativt, alle felter kan bli endret gjennom ekstra hensikter.

Applikasjoner som fungerer med "OI Info": 
* OI Notatblokk
* OI Filbehandler
* OI Handleliste
* OI Konverter CSV
* OI Safe
* OI Nedtelling
* OI Lykt
